Projects Involved

vClavis

vClavis is a cloud based software for the design and analysis of pressure vessels. This software is a competitor to leading software such as VVD and PVElite.

Getting involved in this project I have:

  • Fully developed (front and back-end) a web application for Pressure Vessel Analysis & Design while working closely with an external partner with a Mechanical Engineering background
  • Experience with PHP and Javascript frameworks (Laravel Framework and Vue JS)
  • Worked with database communication
  • Owned a project and helped with the onboarding of new members
  • Implemented the three-dimensional visualization mechanism of this project using the Library Three.js in Javascript

3D Drive

3D Drive is an simple 3D Game that implements a Simulation of a Vehicle with basic steering and driveshaft physics implemented in Javascript using the graphics library Three.js

Some features I have worked on in this project are:

  • Implementation of Ackerman steering for determination of the vehicle’s wheels’ angular position and proper animation of the vehicle rotation around an axis
  • Simulation of conservation of angular momentum between connection of engine and driveshaft using an additionally implemented clutch connection and a gearbox
  • Random map generation with simplified buildings and consecutive road sections of random curvature.
